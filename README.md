# FOLIOFIT 서비스

FOLIOFIT은 디자인, 건축·공간, 연구개발, 개발 분야 지원자가 채용공고와 포트폴리오 PDF를 함께 등록하면 현재 자료에서 확인되는 직무 적합도와 증거 전달력을 분석하고, 프로필 기반 맞춤공고까지 제안하는 모바일 중심 커리어 서비스입니다.

## 구현된 흐름

- 웰컴 및 온보딩
- 디자인 / 건축·공간 / 연구개발 / 개발 분야 선택
- 채용공고 텍스트 입력과 PDF 업로드 UI
- 단계형 AI 분석 진행 화면
- 핏 점수, 요구사항 커버리지, 프로젝트 순서, 페이지 피드백, 문구 수정, 면접 질문
- 이력서와 포트폴리오 프로필 저장
- 저장된 프로필 기반 맞춤공고 제안

## 제품 원칙

- 합격 확률을 예측하거나 표시하지 않습니다.
- 핏 점수는 현재 업로드된 자료의 직무 적합도와 증거 전달력을 뜻합니다.
- 주요 판단은 공고 발췌 또는 포트폴리오 페이지 근거에 연결됩니다.
- 확인되지 않은 역할, 성과, 수치, 경력은 만들지 않습니다.
- 웹에서도 최대 480px 모바일 앱 셸을 유지합니다.

## 실행

```bash
pnpm install
pnpm run dev
pnpm run build
pnpm test
```

## 주요 파일

- `app/page.tsx`: FOLIOFIT 앱 화면과 상태 흐름
- `app/globals.css`: 모바일 앱 셸, 디자인 토큰, 컴포넌트 스타일
- `app/layout.tsx`: 서비스 메타데이터
- `tests/rendered-html.test.mjs`: 서버 렌더링과 제품 원칙 검증
